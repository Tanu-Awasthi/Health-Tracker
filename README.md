# Health Tracker
A Health Tracker application developed in C using file handling. A Health Tracker records vitals of patients. 
## Features
- Add patient records
- Store daily health vitals
- View saved records
## Technologies Used
- C Programming
- File Handling
- Structures
- Arrays
## How to Run
Compile the program:
```bash
 gcc c:\users\admin\desktop\health_tracker.c
```
Run the program:
```bash
.\a.exe
```
## Files
- `health_tracker.c` - Main source code
- `health.txt` - Stores patient data
